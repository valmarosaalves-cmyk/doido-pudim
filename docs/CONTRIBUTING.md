# Contributing

# Issues

When opening an issue, please make sure you:
- Are using the latest Doido Engine 4.0 build (3.x IS NO LONGER SUPPORTED)
- Disclose any modified code you're using
- Follow the templates accordingly
- If you're making a feature request, make sure it is not already added to our Planned Features list

If your issue does not fit any template, please reach out to us on our [Discord Server](https://discord.gg/KVXf9Ygzu5).

# Pull Requests / Contributing Code

When creating a pull request, please make sure of the following:
 - Currently, we are only accepting PRs based on the branch `pudim`. `main` contains the last commit of engine version 3.4.1, which is no longer supported.
 - Please make sure your code is tested and has no conflicts
 - Please make sure your code is formatted and organized correctly (variables should be lowerCamelCase, everything else should be according to our hxformat.json)

If you are unsure what to contribute with, please read our Planned Features list.
